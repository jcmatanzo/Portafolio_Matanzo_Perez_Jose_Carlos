Práctica 2: Página Web Simple

José Carlos Matanzo Pérez
Ingeniería Mecatrónica
Fecha: Agosto 23, 2024
Programación Página Web

- Resumen -

En esta práctica se desarrolló una página web simple utilizando HTML básico. El objetivo principal fue familiarizarse con la estructura fundamental de una página web y aprender a cargar el contenido en línea utilizando GitHub Pages.

La página creada incluye elementos básicos como títulos, texto y una imagen, así como un enlace a una segunda página.

- Introducción -

Una página web es un documento digital diseñado para ser visualizado en navegadores web y formar parte de la World Wide Web. El HTML o Lenguaje de Marcado de Hipertexto es el lenguaje estándar utilizado para estructurar y presentar contenido en la web.

Esta práctica se enfoca en la creación de una página web simple para comprender los fundamentos del desarrollo web y el proceso de publicación en línea.

- Materiales -

1. Ordenador Personal.

2. Editor de texto (Visual Studio Code)

3. Navegador web.

4. Cuenta de GitHub

- Desarrollo -

1. Se creó un nuevo archivo HTML llamado "index.html" utilizando Visual Studio Code.

2. Se implementó la estructura básica del documento HTML, incluyendo las etiquetas "!DOCTYPE html", "html", "head" y "body".

3. Dentro del body, se agregaron elementos como títulos (h1, h2, etc.), párrafos (p), y una imagen (img).

4. Se creó un segundo archivo HTML llamado "pagina2.html" con una estructura similar.

5. Se agregó un enlace en "index.html" para navegar a "pagina2.html".

Simulación

No se requirió propiamente una simulación para esta práctica, ya que el desarrollo se realizó directamente en el editor de texto y se visualizó en el navegador web en tiempo real.

Diseño

El diseño de la página web fue minimalista, enfocándose en la estructura básica y la funcionalidad. Se utilizó la siguiente estructura:

1. Título principal (H1): "INDEX"

2. Subtítulos (H2, H3, H4)

3. Párrafo de texto

4. Imagen

5. Enlace a la segunda página

Construcción

La construcción de la página web se realizó escribiendo el código HTML directamente en el editor de texto. Se utilizaron las siguientes etiquetas principales:

1. "h1", "h2", "h3" y "h4" para los títulos

2. "p" para el texto

3. "img" para la imagen

4. "a" para crear el enlace a la segunda página

- Resultados -

Se logró crear con éxito una página web simple. La página principal (index.html) con títulos, texto e imagen y una página secundaria (pagina2.html) enlazada desde la página principal.

Ambas páginas se visualizan correctamente en el navegador web y sus contenidos se cargaron exitosamente en GitHub Pages y son accesibles en línea.

- Conclusiones -

Esta práctica proporcionó una introducción valiosa a los conceptos básicos del desarrollo web. Se aprendió sobre la estructura fundamental de un documento HTML, cómo agregar diferentes elementos a una página web y cómo crear enlaces entre páginas.
