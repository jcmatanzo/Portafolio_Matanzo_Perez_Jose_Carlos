Proyecto Final: Robot Móvil con Brazo Manipulador

José Carlos Matanzo Pérez
Ingeniería Mecatrónica

José Carlos Valencia Vázquez
Ingeniería Mecatrónica
Fecha: Noviembre 26, 2024
Electrónica Robótica Arduino Bluetooth

- Resumen -

Este proyecto final actúa como la integración de todos los conocimientos adquiridos durante esta y otras materias.

Utilizando conceptos de diseño, mecánica, electrónica y programación, hemos diseñado un robot capaz de moverse sobre superficies planas y recolectar y mover objetos con un brazo manipulador.

- Introducción -

La robótica móvil representa uno de los campos más dinámicos y prometedores en la ingeniería moderna, combinando elementos de mecánica, electrónica y programación para crear soluciones innovadoras a desafíos cotidianos. En este contexto, presentamos el desarrollo de un robot móvil con brazo manipulador, diseñado como proyecto final del curso de Introducción a la Ingeniería Mecatrónica, que demuestra la integración práctica de estos conceptos fundamentales.

Este proyecto materializa la convergencia de tecnologías accesibles y modernas, utilizando la fabricación digital mediante impresión 3D, control electrónico basado en Arduino, conexión vía Bluetooth y una interfaz de usuario móvil para su operación. El robot está diseñado para contar con capacidades tanto de navegación como de interacción con objetos.

Este proyecto no solo sirve como una demostración práctica de conceptos de ingeniería mecatrónica, sino que también explora el potencial de las tecnologías maker y la fabricación digital en el desarrollo de soluciones robóticas accesibles. La combinación de impresión 3D, electrónica de código abierto y desarrollo de aplicaciones móviles representa un enfoque moderno y accesible para la creación de sistemas robóticos funcionales.

- Materiales -

1. Placa Arduino UNO.

2. LAFVIN Expanding Board with Motor Driver Shield.

3. Módulo Bluetooth HC05.

4. Puente H L298.

5. Cable USB para la conexión con el ordenador (Unicamente usado para la carga del código Arduino).

6. Porta pilas de dos pilas tipo 18650.

7. Pilas tipo 18650 de 3.7V, 2000 mAh (2 unidades)

8. Cables jumper macho-macho.

9. Cables jumper macho-hembra.

10. Servomotores MG995 (5 unidades).

11. Motorreductores de doble eje tipo I, 3 Vcc, 1:48 (4 unidades).

12. Ruedas Omnidireccionales 48mm (4 unidades).

13. Tornillería M3.

14. Tornillería M5

16. Filamento PLA

- Desarrollo -

El proyecto fue diseñado en base a las necesidades de movimiento y manipulación de objetos. Utilizando impresión 3D como el método de manufactura de todo el chasis, contamos con un considerable grado de libertad para el diseño realizado en SolidWorks.

El desarrollo del código y la aplicación se basaron en gran medida en los proyectos desarrollados en las prácticas anteriores. Utilizando MIT App Inventor y Arduino IDE, logramos desarrollar un sistema conformado por el código Arduino y una aplicación de móvil. Haciendo uso del módulo HC-05, vinculamos la aplicación con el Arduino vía conexión Bluetooth, permitiéndonos controlar los movimientos y velocidad del robot y su brazo manipulador mediante los botones y diales de la interfaz.

Diseño

El diseño fue realizado en el software de diseño asistido por computadora SolidWorks, utilizando diseños importados de GrabCad para representar componentes más complejos, tales como servomotores y otros componentes electrónicos.

Una vez tomadas las medidas de los componentes electrónicos, diseñamos las piezas con el objetivo de mantener una estructura rígida y resistente pero capaz de realizar los movimientos necesarios. Diseñamos una base amplia para mantener estabilidad en movimiento y con el brazo en una posición extendida.

El código de Arduino y la aplicación fueron desarrollados en conjunto, asegurándonos de mantener coherencia entre ambos códigos para garantizar su compatibilidad. Utilizamos códigos de prácticas anteriores, modificándolos para adecuerlos a las necesidades del mecanismo.

Los servomotores que le brindan movimiento al brazo son controlados por diales que envían un número de identificación para indicar qué servo deseamos mover, al mismo tiempo, envían un valor de posición que le indica al servo a dónde debe moverse, generando así los movimientos deseados.

Los movimientos de las ruedas son controlados por un dial para la velocidad y botones que corresponden a un movimiento específico. Haciendo uso de las ruedas omnidireccionales, podemos realizar movimientos en las cuatro direcciones cardinales, cuatro direcciones en diagonal y, giros hacía la izquierda y derecha sobre el propio eje.

Construcción

Las piezas fueron impresas utilizando filamento PLA, tomando aproximadamente 45 horas de impresión, posteriormente, fueron ensambladas utilizando los servomotores y la tornillería, tomando en consideración la fuerza aplicada en los tornillos para asegurarnos de que estén lo suficientemente justas para mantener firme la estructura, pero manteniendo cierta libertad de movimiento en las articulaciones.

Los componentes electrónicos fueron instalados utilizando tornillería, con las conexiones necesarias conectadas como se muestra en el diagrama de circuitos.

- Resultados -

Las primeras pruebas mostraron un funcionamiento correcto del brazo y algunos movimientos de las ruedas, sin embargo, los movimientos de giro y en diagonal mostraron errores.

Para solucionar los problemas de movimiento de las ruedas, hicimos modificaciones en el código de Arduino y la aplicación. Gracias a estas correcciones logramos un funcionamiento completamente dentro de nuestras expectativas.

- Conclusiones -

Los objetivos del proyecto fueron realizados exitosamente, el robot fue capaz de realizar los movimientos necesarios y manipular objetos correctamente.

Este proyecto sirve como la culminación de la materia de introducción a la ingeniería mecatrónica y nos permitió integrar todas las habilidades desarrolladas en el semestre para realizar un prototipo funcional.
