Arduino Electrónica

- Resumen -

Esta práctica se centró en el uso de la placa Arduino UNO para explorar y comprender el funcionamiento de las salidas digitales. A través de una serie de 9 ejercicios progresivos, se abordaron desde conceptos básicos hasta aplicaciones más complejas, permitiendo una comprensión profunda de cómo controlar diversos componentes electrónicos mediante programación.

- Introducción -

Las salidas digitales son fundamentales en la electrónica y la programación de microcontroladores. Permiten controlar dispositivos externos, como LEDs, motores, relés, entre otros, mediante señales de encendido (HIGH) y apagado (LOW). Esta práctica utiliza el Arduino UNO, una plataforma de desarrollo versátil y accesible, para explorar las capacidades de las salidas digitales en diferentes escenarios.

Para la realización de esta práctica, se utilizaron los siguientes componentes:

2. Cable USB para la conexión con el ordenador

Simulación

Todos los ejercicios fueron simulados utilizando la plataforma Tinkercad. Esta herramienta en línea permite crear circuitos virtuales y programar Arduino, proporcionando un entorno seguro para experimentar antes de la implementación física.

Diseño

En la fase de diseño, se analizó la simulación cada circuito y del código necesario para controlar el Arduino. Analizamos la selección de componentes y la estructuración lógica del programa.

Construcción

1. Se reunieron los componentes necesarios según el diseño.

2. Se montó el circuito en una protoboard, conectando cuidadosamente cada componente.

3. Se cargó el código verificado en la placa Arduino UNO.

4. Se realizaron pruebas para asegurar que el comportamiento físico coincidía con la simulación.

5. Se hicieron ajustes de ser necesarios, tales como la revisión de conexiones o componentes.

- Resultados -

A lo largo de estos 9 ejercicios, se logró una progresión desde el control básico de un solo LED hasta la manipulación de displays de 7 segmentos, demostrando la versatilidad y potencia de las salidas digitales del Arduino UNO.

Cada ejercicio construyó sobre el anterior, permitiendo una comprensión acumulativa de los conceptos y técnicas de programación necesarios para controlar eficientemente las salidas digitales.
